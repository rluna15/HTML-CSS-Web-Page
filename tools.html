Software and Hardware

List of tools you'll need to get you started.

Hardware

Computer
Nothing to special is needed to get started. For 2D artists, a simple computer from your local best buy should do. As for 3D artists then a bit more powerfull computer is needed. Although work can be done without a dedicated graphics card. It would be best to have a computer with one for better performance.
Wacom Tablets
Wacom are a good choice to start off. Don't spend money on a cintiq, since they are way to expensive.
Huion Tablets
Huion are a cheaper version of wacom but they are just as good. Very good choice if you are tight on money.

Software For 2D

Photoshop
Photoshop is the industrial standard but adobe sells it as a service. There is a monthy fee but with that fee they offer cloud storage.
Clip Studio Paint
Clip Studio was mainly made for comic books but works the same way as photoshop with out the service fee. Sometimes it's on sale.

Software For 3D

Blender
Free software for 3D modeling, a great start for anyone to learn.
ZBrush
3D modeling software that is mainly focused in 3D sculpting. Mostly used for characters.
